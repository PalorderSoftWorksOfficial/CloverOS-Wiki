About CloverOS

CloverOS is a custom Minecraft operating system designed for the CC:Tweaked mod and CraftOS. It draws inspiration from Minux and Windows 10/11, combining the lightweight structure of Minux with a modern and user-friendly design inspired by Windows.

CloverOS is a complete rewrite built to feel like a desktop environment inside Minecraft. Originally based on Minux concepts, it has since been redesigned and reimplemented independently.

Note: Previous issues with the file scraper while examining Minux have been resolved.

Features

  • Windows-style interface and experience
  • Lightweight and optimized for in-game performance
  • Fully compatible with CC:Tweaked and CraftOS
  • Modular design allowing easy expansion and updates
  • Open for community feedback and contributions

Installation

To install CloverOS, open an Advanced Computer (or CraftOS terminal) in Minecraft and enter the following command:

wget run https://cloveros.madefor.cc/netinstall.lua

The installer will automatically download and set up CloverOS. You need to modify some configurations:

  1. In your Minecraft world, open the Mods menu and select CC:Tweaked, then click Config.
  2. Select the file sorted in the Server Configurations and click Modify.
  3. Set Computer Space Limit to 1073741824 exactly.
  4. Do the same for Floppy Space Limit.
  5. In the Turtle folder, click ON for Need Fuel.

These instructions are only for CC:Tweaked and will be improved in the upcoming wiki.
